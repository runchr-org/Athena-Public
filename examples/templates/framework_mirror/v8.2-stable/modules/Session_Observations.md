# Session_Observations.md — Calibration Journal (Template)

> **What this is**: A longitudinal calibration journal. Athena appends dated observations
> about the user's decision patterns, pricing accuracy, and execution behavior across sessions.
> Over time it becomes the evidence base for personalized calibration (e.g., "you consistently
> underprice long-form work by N%").
>
> **Why this copy is empty**: The author's live journal contains personal pricing and client
> data and is excluded from the public release per the allowlist deploy model
> (see `examples/workflows/deploy.md`). The structure below is the reusable part.

---

## Entry Format

```markdown
- **[Pattern Name]**: Observed behavior, with per-instance evidence
  ([Project A]: delta, [Project B]: delta). Quantified impact where possible.
- **Context**: What was happening when the pattern emerged.
- **Insight (S)**: The strategic takeaway, with a confidence marker (🔵 0.9 / 🟢 0.7 / 🟡 0.5).
- **Connects to**: Cross-references to sessions, protocols, or case studies.
```

## Calibration Principles

- **Chat Logs > Memory**: Verify empirical claims (pricing, dates, commitments) from primary
  sources, not recalled notes.
- **Quantify the delta**: "Underpriced" is an opinion; "-47% vs market across 3 instances" is data.
- **Confidence markers**: Every insight carries an explicit confidence level that updates as
  evidence accumulates.
